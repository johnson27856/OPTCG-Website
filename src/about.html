About This Website

This website is the result of a side project that I have been meaning to make. Growing up, I was fascinated by trading card games like Pokémon and I loved collecting cards of my favorite pocket monsters. Even now, as a soon-to-be college graduate, I still find myself looking online for good deals on Pokémon booster packs so that I can relive some of my childhood memories.

The reason I decided to make this website about One Piece instead of Pokémon is because I have more friends who enjoy playing the One Piece TCG than those who enjoy playing the Pokémon TCG. It was honestly a hard decision to make at first, but I figured that once I could confidently program one, it would be a lot easier programming the other later down the road.

If it wasn't obvious already, this website is still in heavy development! I am constantly trying to implement new features, so be sure to check back every once in a while to see if anything new has been added! I do log the changes too, so you can see all the website version updates in the "News" tab on the top navigational bar. In any case, thanks for visiting my website! 😎
