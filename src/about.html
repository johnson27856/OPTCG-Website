About This Website

This website is a side project that I had been meaning to make for a while now. Growing up, I was fascinated by trading card games like Pokémon and I loved collecting cards of my favorite pocket monsters. Even now, as a soon-to-be college graduate, I still find myself looking online for good deals on Pokémon booster packs so that I can relive some of my childhood memories.

The reason I decided to make a website about One Piece instead of Pokémon is simply because I have more friends who play the One Piece TCG than those who play the Pokémon TCG. It was a difficult decision to make initially, but I figured that once I could get a feel for programming one website, it would be a lot easier programming another later down the road.

If it wasn't obvious already, this website is still in HEAVY development! I am constantly trying to implement new features, so be sure to check back every once in a while to see if anything new has been added! I log all of the changes too, so you can see the most recent website version updates in the "News" tab on the top navigational bar. I also have a GitHub repo for the website that includes a changelog which provides a detailed history of all the changes.

In any case, I hope you have a fantastic day and thanks for visiting my website! 😎
